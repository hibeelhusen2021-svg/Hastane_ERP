# 🧠 MindCare ERP & CPM Enterprise SaaS

MindCare, psikolojik danışmanlık merkezlerinin operasyonel maliyetlerini azaltmak ve yönetim süreçlerini optimize etmek amacıyla geliştirilmiş B2B dikey bir SaaS (Software as a Service) girişimidir.

## 🚀 Girişimcilik & Proje Yönetimi Özellikleri
- **Matematiksel Proje CPM (Kritik Yol Yöntemi):** Projenin pazara çıkış süresini (Time-to-Market) optimize eden, bolluk (slack) sürelerini hesaplayan analitik matris.
- **Finansal Bütçe & Nakit Akışı Yönetimi:** Girişimin sermaye erime oranını (Burn Rate) takip eden dinamik finansal loglama sistemi.
- **Ölçeklenebilir İlişkisel Veritabanı:** Normalizasyon kurallarına tam uyumlu, şube-oda-danışan ilişkilerini Foreign Key yapılarıyla bağlayan RDBMS mimarisi.

## 🛠️ Kurulum ve Çalıştırma
Proje hiçbir harici kütüphane bağımlılığı (dependency) gerektirmeden saf (Vanilla) HTML5, CSS3 ve JavaScript mimarisiyle geliştirilmiştir. 

1. Bu depoyu klonlayın: `git clone https://github.com/KULLANICI_ADIN/mindcare-erp-saas.git`
2. Klasör içindeki `index.html` dosyasına çift tıklayarak sistemi tarayıcınızda canlı olarak çalıştırabilirsiniz.
